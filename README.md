# 内嵌的临时Tomcat对照项目

本项目为一个 **临时性 Tomcat 项目**，其存在目的仅为作为 **父 Spring Boot 项目** 的等价对照物（即用于功能、行为或性能的对比验证）。

## 项目性质
- **类型**：传统 Servlet 容器（Tomcat）部署结构
- **生命周期**：临时
- **对标项目**：[ujsp](https://github.com/frycod128/ujsp)

## 关于更新
当对应的 Spring Boot 项目发生更新时：
1. **本项目将被归档** (内容部分，环境不会)
2. 归档方式：压缩为 `.zip` 或 `.tar.gz` 文件，存放于 `archive/` 目录下，并按日期命名（例如 `archive/project_temp_YYYYMMDD.zip`）
3. 归档后，项目将更新作为新需求的活跃开发环境使用

## 归档说明
归档文件包含此声明文件及完整项目源码/配置，以备后续追溯或对照验证。归档后如需再次对照，请基于对应版本的 Spring Boot 项目重新生成临时对照环境。
